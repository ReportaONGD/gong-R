¡Bienvenido a la instalación de demostración de GONG 2.0!
Desde aqui podrá acceder a una instalación en pruebas y a modo de demostracion de la version 2.0 de GONG.
(Última actualización Octubre 2010)

  • Para ver la documentación tecnica, de usuario, u otros recursos visite http://www.gestionong.es
  • La actual versión de GONG esta probada para Firefox 3.6. Si tiene cualquier versión anterior de firefox o si utiliza otros navegadores puede ser que algunos detalles de diseño no se muestren adecuadamente.
  • Cualquier información que se introduzca en el sistema se borrará diariamente.
  • La actual instalación contiene una serie de datos de prueba sencillos. Dichos datos son restaurados diariamente.
  • El usuario / contraseña para acceder al sistema es: "admin" / "gong". Este usuario tiene permisos de administración y tiene habilitadas todas las secciones.
